# apteka-app
Мобильное приложение для заказа лекарств

## Команда
- [Тоболин Д.А] — разработчик серверной части, технический писатель
- [бРАГИН К.] — разработчик клиентской части, разработчик БД, дизайнер, тестировщик

## Прецеденты (3)
1. Посмотреть список лекарств
2. Добавить лекарство в корзину
3. Оформить заказ

## Технологии
- Клиент: Android Kotlin
- Сервер: C# ASP.NET Core
- База данных: SQL Server
